Interactive data visualization is the use of tools and processes to produce a visual representation of data which can be explored and analyzed directly within the visualization itself. This interaction can help uncover insights which lead to better, data-driven decisions. Interactive data visualization refers to the use of interactive visual elements, such as charts, graphs, and maps, to display and analyze data in a way that allows users to manipulate and explore the data in real-time. This approach allows users to quickly understand patterns and trends in the data, and make more informed decisions based on the insights gained.


In this exercise, the given data presents Daily Emissions for Airports in France. The data contained 22064 rows in a csv file format and there were no empty or invalid cells. In the following exercises, Vega-lite.js has been used to make visualizations from data to better understand its different aspects and to perform analytics. Some popular tools for interactive data visualization include Tableau, and Plotly.
